# Recibos

Recibos
=======

16truenonelisttrue

Para acessar esse módulo no sistema, basta seguir os seguintes passos:

Um recibo é um documento que comprova o recebimento de um pagamento ou a entrega de um bem ou serviço. Ele registra informações importantes para a transparência e segurança da transação financeira. Recibos são fundamentais para manter um registro formal de transações e evitar eventuais disputas ou dúvidas.

Menu Financeiro > Recibos

* * *

Será apresentada a seguinte tela:

Tela geral do módulo de recibos

No sistema, existem duas opções para a geração de recibos, conforme a necessidade da transação:

1.  **Recibo de Faturas**  
    Esta opção permite gerar recibos com base em faturas de convênio, apresentadas no módulo Faturas de Convênio. É ideal para quando se deseja emitir recibos vinculados diretamente a essas faturas.
    
2.  **Recibo Avulso**  
    Esta opção permite a geração de um recibo para uma transação isolada, inserindo um valor específico para um determinado cliente. O **Recibo Avulso** é útil para registrar pagamentos não vinculados diretamente a faturas de convênio.
    

Ambas as opções garantem flexibilidade na emissão de recibos conforme o tipo de transação e necessidade do usuário.

Recibo de Fatura
----------------

Opções para recibo de fatura

A opção `Cliente` oferece a possibilidade de filtrar faturas por clientes específicos, facilitando a seleção das faturas desejadas. A seleção de cliente, no entanto, não é obrigatória.

Clique no botão para escolher as faturas que deseja incluir na geração do relatório de recibo. Ao clicar, será exibida a seguinte tela:

Tela para seleção das faturas

Na parte superior da tela, estão disponíveis filtros para facilitar a busca e a localização das faturas específicas:

*   `Data Inicial` e `Data Final`: Filtra as faturas dentro de um intervalo de datas específico.
    
*   `Consultar Data`: Define qual data deve ser considerada na consulta (`De faturamento` ou `De vencimento`).
    
*   `Fatura`: Permite buscar uma fatura específica pelo seu código.
    

Na parte inferior do box, há dois resumos que facilitam a visualização rápida dos valores e quantidades de **faturas listadas** e **selecionadas**.

Após configurar os filtros desejados, clique no botão para atualizar a listagem de faturas de acordo com os critérios aplicados. Cada fatura apresenta informações essenciais, como o `Fatura`, `Cliente`, `Vencimento` e `Valor (R$)`.

Ao lado de cada fatura listada, há uma caixa de seleção que pode ser marcada para incluir a(s) fatura(s) na geração do relatório.

Após selecionar a(s) fatura(s) desejada(s), clique no botão . O sistema retornará para a tela inicial do módulo. Clique no botão para gerar o relatório contendo as informações das faturas selecionadas.

Recibo Avulso
-------------

Opções para recibo avulso

A opção `Cliente` oferece a possibilidade de filtrar um cliente específico para a emissão de um **recibo avulso**. No entanto, essa seleção não é obrigatória. Caso nenhum cliente seja selecionado, o recibo será gerado para todos os clientes cadastrados.

Informe o `Valor` que será apresentado no recibo. Se necessário, insira uma `Mensagem Adicional` que será exibida no relatório gerado. Após preencher as informações necessárias, clique no botão para gerar o relatório contendo os dados do recibo avulso.
