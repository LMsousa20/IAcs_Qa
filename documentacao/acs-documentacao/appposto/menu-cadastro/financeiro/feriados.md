16truenonelisttrue

Feriados
========

O módulo de **Cadastro de Feriados** permite cadastrar e gerenciar datas que não devem ser consideradas como dias úteis para operações financeiras, como o recebimento de valores ou compensações de pagamentos, especialmente aqueles realizados via cartão. Essa configuração é importante para ajustar corretamente os prazos de liquidação e garantir que o sistema respeite as particularidades de cada data, evitando inconsistências no fluxo financeiro.

Para acessar esse módulo no sistema, basta seguir os seguintes passos:

Menu Cadastro > Financeiro > Feriados

Será apresentada a seguinte tela:

Tela de Listagem de Feriados

Ao acessar o módulo, a tela principal será exibida, permitindo a visualização e o gerenciamento dos feriados já configurados. Esta tela é organizada para facilitar a navegação e a inserção de dados.

As principais funcionalidades são:

**Adicionar Novo Feriado**

Permite o cadastro de um novo feriado no sistema, garantindo que todos os feriados relevantes estejam devidamente registrados.

**Filtro de Pesquisa**

Facilita a busca do feriado específico pelo nome, agilizando a localização das informações necessárias.

**Edição e Visualização**

Permite modificar ou consultar os detalhes de um feriado já cadastrado, assegurando a atualização constante dos dados.

Apresentação da tela e campos
-----------------------------

Ao clicar no botão , a seguinte tela será apresentada:

### Box 1 - Novo Feriado

Box Novo Feriado

*   `Dia/Mês`: Campo para indicar a data que será considerada feriado no sistema, no formato ddMM.
    
*   `Motivo`: Descrição breve do feriado.
    

* * *

Como Funciona (Passo a Passo)
-----------------------------

Esta seção detalha os procedimentos para incluir, editar e excluir feriados no sistema.

### Incluir Feriado

Para cadastrar um novo feriado no sistema, siga os passos abaixo:

Acesse o módulo de Feriados através do caminho: `Menu Cadastro > Financeiro > Feriados`.

Na tela de listagem dos feriados, clique no botão .

Preencha os campos no box **Novo Feriado**:

*   `DIA/MÊS`: Indique a data com dia e mês (4 dígitos).
    
*   `MOTIVO`: Insira o nome do feriado.
    

Após preencher todos os campos e revisar as informações, clique no botão . A tela será redirecionada para a listagem dos feriados.

* * *

### Editar Feriado

Para modificar os dados de um feriado já cadastrado, siga os passos:

Acesse o módulo de Feriados. A tela de listagem exibirá todos os feriados cadastradas.

Localize o feriado que deseja editar na lista. Você pode usar o filtro de pesquisa para agilizar a busca.

Clique no ícone ao lado do feriado desejado.

A tela de cadastro do feriado será aberta, permitindo a modificação dos campos.

Após realizar as modificações necessárias, clique no botão para aplicar as alterações.

* * *

### Excluir Feriado

Para remover um feriado do sistema, siga os passos:

Acesse o módulo de Feriados.

Localize o feriado que deseja excluir na tela de listagem.

Clique no ícone de ao lado do feriado desejado.

Uma tela de confirmação será exibida, solicitando que você confirme a exclusão.

Clique no botão para finalizar o processo de exclusão.
