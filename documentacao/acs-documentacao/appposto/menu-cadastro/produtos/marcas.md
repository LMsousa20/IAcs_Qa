Marcas
======

O módulo de **Cadastro de Marcas** permite registrar e gerenciar as marcas comerciais associadas aos produtos do sistema, garantindo a padronização das informações cadastradas. Esse recurso oferece maior controle sobre o portfólio de produtos, além de facilitar buscas, filtros e análises em relatórios.

Para acessar esse módulo no sistema, basta seguir os seguintes passos:

Menu Cadastro > Produtos > Marcas

Será apresentada a seguinte tela:

Tela de Listagem de Marcas

Ao acessar o módulo, a tela principal será exibida, permitindo a visualização e o gerenciamento das marcas já configuradas. Esta tela é organizada para facilitar a navegação e a inserção de dados.

As principais funcionalidades são:

**Adicionar Nova Marca**

Permite o cadastro de uma nova marca no sistema, garantindo que todas as marcas relevantes estejam devidamente registradas.

**Filtro de Pesquisa**

Facilita a busca da marca específica pelo nome, agilizando a localização das informações necessárias.

**Edição e Visualização**

Permite modificar ou consultar os detalhes de uma marca já cadastrada, assegurando a atualização constante dos dados.

Como Funciona (Passo a Passo)
-----------------------------

Esta seção detalha os procedimentos para incluir, editar e excluir marcas no sistema.

### Incluir Marca

Para cadastrar uma nova marca no sistema, siga os passos abaixo:

Acesse o módulo de Marcas através do caminho: `Menu Cadastro > Produtos > Marcas`.

Na tela de listagem das marcas, clique no botão .

Preencha os campos no box **Nova Marca**:

*   `Descrição`: Insira o nome da marca.
    

Após preencher todos os campos e revisar as informações, clique no botão . A tela será redirecionada para a listagem das marcas.

* * *

### Editar Marca

Para modificar os dados de uma marca já cadastrada, siga os passos:

Acesse o módulo de Marcas. A tela de listagem exibirá todos as marcas cadastradas.

Localize a marca que deseja editar na lista. Você pode usar o filtro de pesquisa para agilizar a busca.

Clique no ícone ao lado da marca desejada.

A tela de cadastro da marca será aberta, permitindo a modificação do campo.

Após realizar as modificações necessárias, clique no botão para aplicar as alterações.

* * *

### Excluir Marca

Para remover uma marca do sistema, siga os passos:

Acesse o módulo de Marcas.

Localize a marca que deseja excluir na tela de listagem.

Clique no ícone de ao lado da marca desejada.

Uma tela de confirmação será exibida, solicitando que você confirme a exclusão.

Clique no botão para finalizar o processo de exclusão.

O sistema **não permitirá a exclusão** de uma marca caso ela esteja associada a um produto. Esta medida de segurança garante a integridade dos dados e evita inconsistências no sistema.
