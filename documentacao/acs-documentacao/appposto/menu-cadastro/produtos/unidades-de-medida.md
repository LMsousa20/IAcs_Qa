# Unidades de Medida

16truenonelisttrue

Unidades de Medida
==================

O módulo de **Cadastro de Unidades de Medida** permite registrar e gerenciar as unidades utilizadas para comercialização, controle e movimentação de produtos no sistema, garantindo a padronização de informações em relatórios, notas fiscais e no controle de estoque. Cada unidade de medida define a forma como um produto será vendido, comprado ou controlado no estoque — seja por **quantidade**, **peso**, **volume**, entre outros.

Para acessar esse módulo no sistema, basta seguir os seguintes passos:

Menu Cadastro > Produtos > Unidades de Medida

Será apresentada a seguinte tela:

Tela de Listagem de Unidades

Ao acessar o módulo, a tela principal será exibida, permitindo a visualização e o gerenciamento das unidades já configuradas. Esta tela é organizada para facilitar a navegação e a inserção de dados.

As principais funcionalidades são:

**Adicionar Nova Unidade**

Permite o cadastro de uma nova unidade de medida no sistema, garantindo que todas as unidades relevantes estejam devidamente registradas.

**Filtro de Pesquisa**

Facilita a busca da unidade específica pelo nome, agilizando a localização das informações necessárias.

**Edição e Visualização**

Permite modificar ou consultar os detalhes de uma unidade já cadastrada, assegurando a atualização constante dos dados.

Como Funciona (Passo a Passo)
-----------------------------

Esta seção detalha os procedimentos para incluir, editar e excluir unidades no sistema.

### Incluir Unidade de Medida

Para cadastrar uma nova unidade no sistema, siga os passos abaixo:

Acesse o módulo de Unidades de Medida através do caminho: `Menu Cadastro > Produtos > Unidades de Medida`.

Na tela de listagem das unidades, clique no botão .

Preencha os campos no box **Nova Unidade**:

*   `Descrição`: Insira o nome da unidade.
    
*   `Símbolo`: Insira o símbolo que representa a unidade.
    

Após preencher todos os campos e revisar as informações, clique no botão . A tela será redirecionada para a listagem das unidades.

* * *

### Editar Unidade de Medida

Para modificar os dados de uma unidade já cadastrada, siga os passos:

Acesse o módulo de Unidades de Medida. A tela de listagem exibirá todos as unidades cadastradas.

Localize a unidade que deseja editar na lista. Você pode usar o filtro de pesquisa para agilizar a busca.

Clique no ícone ao lado da unidade desejada.

A tela de cadastro da unidade será aberta, permitindo a modificação dos campos.

Após realizar as modificações necessárias, clique no botão para aplicar as alterações.

* * *

### Excluir Unidade de Medida

Para remover uma unidade do sistema, siga os passos:

Acesse o módulo de Unidades de Medida.

Localize a unidade que deseja excluir na tela de listagem.

Clique no ícone de ao lado da unidade desejada.

Uma tela de confirmação será exibida, solicitando que você confirme a exclusão.

Clique no botão para finalizar o processo de exclusão.

O sistema **não permitirá a exclusão** de uma unidade caso ela esteja associada a um produto. Esta medida de segurança garante a integridade dos dados e evita inconsistências no sistema.
