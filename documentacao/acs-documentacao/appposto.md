# AppPosto

Acesso ao sistema
-----------------

Para acessar o sistema _AppPosto_, é necessário entrar no seu navegador de internet e acessar o link respectivo ao seu posto, onde após o ‘_/login/_ ‘haverá o ID do seu sistema, como por exemplo:

html

A tela de login será apresentada:

Tela inicial de login

Para realizar o primeiro acesso ao sistema, utilize a senha do usuário padrão **admin**, que será fornecida após a criação do ambiente.

Após o login inicial, você poderá editar as informações do usuário padrão e cadastrar novos usuários, definindo suas respectivas senhas e permissões de acesso.

Se o banco de dados possuir múltiplas empresas, o campo **Empresa** exibirá todas as empresas disponíveis para login. O usuário poderá selecionar a empresa desejada, desde que tenha sido previamente cadastrado para acessar essa empresa específica.

Tela inicial
------------

Ao acessar o sistema, você será direcionado à tela inicial, que é composta por três elementos principais: a barra lateral de menus, a barra de status e o quadro de informações. Abaixo, você encontrará uma descrição detalhada de cada um desses componentes:

Tela inicial do sistema após login

### Barra lateral de menus

Localizada à esquerda da tela, essa barra contém todos os menus de opções do sistema. Através dela, você pode navegar pelas diferentes funcionalidades e módulos do sistema.

*   Dashboard
    
*   Menu Cadastro
    
*   Menu Estoque
    
*   Menu Financeiro
    
*   Menu Movimento
    
*   Menu Opções
    
*   Menu Relatórios
    

### Barra de status

Posicionada na parte inferior da tela, a barra de status exibe informações importantes sobre a sessão atual. Nela, você encontrará:

*   **Nome do Grupo**: Identifica o grupo ou unidade organizacional que está sendo acessado no momento.
    
*   **Usuário**: Mostra o nome do usuário que está logado no sistema.
    
*   **Empresa**: Exibe o nome da empresa associada à sessão ativa.
    
*   **Versão**: Indica a versão atual do sistema que está em uso, permitindo que você saiba se está trabalhando com a versão mais recente.
    

### Quadro de informações

Este quadro está localizado em uma área de destaque na tela inicial e oferece um resumo das informações sobre o usuário e a sessão:

*   **Nome do Usuário**: Exibe o nome do usuário atualmente logado.
    
*   **Nome da Empresa**: Informa a empresa à qual o usuário está vinculado.
    
*   **Data do Último Acesso**: Mostra a última data e hora em que o usuário acessou o sistema, proporcionando um controle adicional de segurança.
    
*   **Fuso Horário**: Indica o fuso horário em que o sistema está configurado, essencial para a correta interpretação das datas e horários registrados.
